DOT BASH

This area sets up the interactive bash session for all accounts.

Table of Contents

  • 2 How it works

1 Installation

Run ../install.sh in a new account. It creates .bashlogin, .bashlogout and .bashrc as symlinks to files of the same, non-dotted names in this directory. If any of these files exists, they will be moved asside. If .profile or .bashprofile exist they will be moved aside.

2 How it works

The usual rc files will load all relevant configuration chunks found under 

~/dot/bash/{login,logout,rc}/DIR/*

Where DIR is expanded in turn to to be the string "any", followed by 

uname -s             # Kernel name (Linux)
lsb_release -s -i    # Distribution name (Ubuntu, Debian)
lsb_release -s -c    # Distribution codename (maverik, squeeze)
hostname -d          # Domain name
uname -n             # Hostname

Interactive logins first source ~/.bashrc followed by any relevant chunks under login/.

Individual script chunks can make use of functions defined in 

~/dot/bash/common.sh
~/dot/bash/common/*
